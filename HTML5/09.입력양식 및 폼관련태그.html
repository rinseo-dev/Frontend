form태그내에 submit버튼 클릭시
해당 form안에 사용자가 작성한 값들을 서버로 전송하여 요청하는 역할 수행

form action="서버로전송할경로" method="get | post"

* action속성 : 해당 form에 사용자가 입력한 값을 전달하면서 요청할 서버의 경로 제시 * method속성 : 요청 전송 방식을 지정하는 속성( get:기본값 | post ) > get 방식 : 요청시 사용자가 입력한 값들이 url에 노출되는 방식
> post 방식 : 요청시 사용자가 입력한 값들이 url에 노출되지 않는 방식
form내의 submit버튼 클릭시 action에 지정한 서버로

사용자가 입력한 값들이 넘어가면서 요청됨
(사용자가 넣은 값은 key=value 세트로 전달)
