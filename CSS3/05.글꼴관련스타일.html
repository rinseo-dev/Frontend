font-family : 글꼴을 지정할 때 사용

        선택 { font-family:글꼴명; }
        
        ** {font-family: A폰트, B폰트, C폰트, ...}
            글꼴은 ','를 넣어서 설정할 수 있다. 이 경우 A폰트가 없으면 B, B도 없으면 C가 적용되며
            설정한 폰트가 모두 없을 경우 기본값으로 정의된 폰트로 설정된다

font-size : 글꼴의 크기를 변경할 때 사용

        선택자 { font-size:크기( px | em | rem | % ) }

        ** px : 고정된 크기 / em : 상위 글꼴 크기 기준 / rem : 최상위 글꼴 크기 기준
            px, rem이 많이 사용된다고 함
  • 고정크기 px테스트
  • 가변크기 em
  • 가변크기 %

font-wieght : 글꼴의 두께 지정

        선택자{
            font-weight : normal | bold | bolder | lighter | 100 | 400 | 900;
        }
  • 원래 굵기
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 가늘게

font-varient : 영문텍스트 문구를 작은 대문자로 변경할 때 사용

선택자{ font-variant : small-caps; }
