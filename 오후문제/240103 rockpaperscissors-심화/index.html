GAME RULE ! X

1. 게임이 시작 되면 가위,바위,보 중 하나를 선택합니다.

2. 선택된 순간 랜덤으로 컴퓨터에서 가위,바위,보 중 하나가 선택됩니다.

3. 알림창이 뜨고 확인을 누르면 PC, USER 중 승자는 10점을 가져갑니다. 비겼을 시 DRAW가 1씩 증가합니다.

4. PC, USER 중 50점을 넘긴 사람이 승자가 됩니다.

5. 승자가 결정되면 점수와 횟수가 나오고 게임을 다시 시작할껀지 끝낼껀지 창이 뜹니다.
